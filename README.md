### 라이센스 본문

```
G마켓 브랜드 아이덴티티를 담은 서체 'Gmarket Sans'를 소개합니다. 
 
Gmarket Sans는 다양한 브랜드, 판매자, 사용자가 함께 성장하며 가치를 나눌 수 있도록 만든 글꼴입니다. 
 
누구나 아무 제약 없이 쉽게 사용할 수 있습니다. 
 
Gmarket Sans는 누구나 제약 없이 자유롭게 수정하고 재배포 하실 수 있습니다.
```
